# EEG Preprocessing Pipeline for P300 ERP Analysis

**Development and reproduction of a standard EEG preprocessing pipeline for P300 event-related potential (ERP) analysis using MNE-Python.**

> Bi-Weekly Project · July 20 – August 3, 2026

## Summary

This project builds a complete EEG preprocessing and analysis pipeline to reproduce the
classical **P300 event-related potential** from the [BNCI Horizon 003-2015 Visual P300 Speller
dataset](https://bnci-horizon-2020.eu/database/data-sets) using
[MNE-Python](https://mne.tools/stable/index.html). It is designed to introduce the team to EEG
signal processing, ERP analysis, and reproducible computational neuroscience workflows.

## Goal

To reproduce the classical P300 event-related potential from the BNCI Horizon 003-2015 Visual
P300 Speller dataset using MNE-Python, while developing a complete EEG preprocessing and analysis
pipeline. The project introduces our team to EEG signal processing, ERP analysis, and reproducible
computational neuroscience workflows.
