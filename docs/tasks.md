# Proposed Tasks

The workflow is broken into five sequential tasks. Links to supporting material are collected in
[References](references.md).

## Task 1 — Literature Review

- Understand the P300 component and the oddball paradigm.
- Review the Farwell & Donchin paper. ([reference](references.md#papers))
- Review Christoph Guger's publication on P300-based BCI data acquisition. ([reference](references.md#papers))

## Task 2 — Dataset Familiarization

- Download and inspect the BNCI Horizon 003-2015 dataset. ([reference](references.md#dataset))
- Understand the experimental design, event markers, and recording protocol.

## Task 3 — EEG Preprocessing

- Load the data using MNE-Python.
- Filtering.
- Event extraction.
- Epoching.
- Baseline correction.
- Artifact inspection.

## Task 4 — ERP Analysis

- Compute averaged ERPs.
- Compare target vs. non-target responses.
- Identify the P300 peak.
- Visualize the results.

## Task 5 — Interpretation & Presentation

- Discuss observations.
- Document the workflow.
- Present findings and challenges.
